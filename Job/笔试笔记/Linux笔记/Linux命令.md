# Linux命令总结

## 重定向

|        |  符号  |  含义  |
|:-------|:------:|:-------|
||>|输出重定向到一个文件或设备，覆盖原来的文件|
||>!|输出重定向到一个文件或设备，强制覆盖原来的文件|
||>>|输出重定向到一个文件或设备，追加原来的文件|
||<|输入重定向到一个程序|

## 查看信息

|        |  符号  |  含义  |
|:-------|:------:|:-------|
|        |cat /proc/cpuinfo|查看一台linux机器的CPU信息|
||du|查看该目录占用文件系统数据块的情况|
||cat /proc/swaps|查看SWAP分区信息|
||df -lh|查看硬盘信息|
